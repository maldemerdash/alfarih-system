تعديل v101:
- توحيد رفع صور ومرفقات المنشأة إلى Supabase Storage بدل حفظ الصور داخل المتصفح.
- شعار المنشأة، خلفية تسجيل الدخول، ومرفق العنوان الوطني تحفظ الآن كملفات فعلية مع حفظ معرف المرفق فقط في إعدادات المنشأة.
- منع عودة صور base64 الثقيلة إلى app_state لتسريع تحميل الموقع وتقليل حجم المزامنة.
- رفع المرفقات الجديدة إلى Storage فقط، وعدم حفظها محليًا إذا تعذر الاتصال بالسحابة حتى لا تختلف البيانات بين الأجهزة.
- إضافة تحسينات كاش للروابط الموقعة وتقليل إعادة تحديث صورة الهيدر.

ملاحظة: لا يوجد SQL جديد إذا كانت جداول/حاويات المرفقات السابقة موجودة كما في النسخ السابقة.
