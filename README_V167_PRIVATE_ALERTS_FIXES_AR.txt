إصلاحات v167 - التنبيهات الخاصة
=================================

هذه النسخة تعالج ملاحظات التنبيهات الخاصة بعد إضافة v166.

ما تم إصلاحه:
- عند الضغط على إنهاء في جدول التنبيهات الخاصة يصبح التنبيه منتهيًا، ويختفي زر إنهاء وزر تأجيل، ويبقى زر حذف فقط.
- عند حذف تنبيه لا يتم فقدانه نهائيًا؛ يتم حفظه كتنبيه محذوف مع تاريخ الحذف واسم المستخدم الذي قام بالحذف.
- تمت إضافة جدول أسفل صفحة التنبيهات الخاصة باسم التنبيهات المحذوفة.
- جدول التنبيهات المحذوفة يعرض: الموضوع، النص، تاريخ الإنشاء، تاريخ التذكير، مدة التذكير قبل، الموظف المرتبط، تاريخ الحذف، ومن قام بالحذف.
- تم تثبيت ترتيب أيقونة التنبيهات الخاصة وجرس الإشعارات أعلى الموقع، بحيث لا تتبادل الأيقونات أماكنها عند الضغط.
- عند إنشاء تنبيه خاص مرتبط بموظف يتم تسجيله داخل ملاحظات الموظف.
- عند تأجيل أو إنهاء أو حذف التنبيه المرتبط بموظف يتم تحديث نفس ملاحظة الموظف المرتبطة بالتنبيه، بدون تكرار سجلات الملاحظات.

مهم:
- تم الإبقاء على مفتاح التخزين المحلي:
  nawah-private-alerts-v166
- وتم الإبقاء على حقل الحفظ السحابي:
  privateAlertsV166
- السبب هو حماية بيانات التنبيهات الموجودة وعدم إنشاء مخزن جديد فارغ عند التحديث.

الاختبارات:
- فحص JavaScript عبر node --check.
- اختبار متصفح محلي على Google Chrome.
- اختبار حالة التنبيه المنتهي.
- اختبار الحذف وجدول المحذوفات.
- اختبار ثبات ترتيب أيقونتي الإشعارات والتنبيهات الخاصة.
- اختبار تسجيل التنبيه المرتبط بموظف داخل ملاحظات الموظف.
