نسخة v144 - تنظيف واستقرار وتسريع

الأساس: نسخة v143.

ما تم تنفيذه:
1. الحفاظ على نفس الشكل والمنطق الحاليين بدون تغيير الواجهة أو مسارات العمل.
2. تحديث كاش الملفات إلى v144:
   - style.css?v=144
   - app.js?v=144
3. إضافة طبقة أداء خفيفة وآمنة:
   - تسريع الوصول للموظف عبر فهرس داخلي بدل البحث المتكرر داخل القائمة.
   - تخفيف تكرار جلب ملفات/حسابات المستخدمين بصلاحية TTL قصيرة.
   - موازنة حفظ الحالة السحابية حتى لا تتم عمليات حفظ متلاحقة تسبب بطء.
   - توحيد إعادة الرسم داخل إطار واحد لتقليل ثقل الواجهة.
4. تنظيف الحزمة النهائية من ملفات README القديمة المتراكمة؛ بقيت ملفات التشغيل الأساسية فقط.
5. عدم تعديل منطق الصلاحيات، الإشعارات، المالية، الإجازات، المرفقات، أو التقارير.
