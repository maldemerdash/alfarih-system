نسخة v163 - إصلاح أخطاء التشغيل
================================

تم اختبار الصفحة فعليًا عبر Chrome headless وقراءة رسائل الكونسول.

الإصلاحات:
1. منع حفظ حالة التطبيق في Supabase قبل تسجيل الدخول.
2. إيقاف ظهور أخطاء 401/RLS الناتجة عن الحفظ السحابي المبكر من شاشة الدخول.
3. إضافة favicon من `sar-symbol.png` لمنع طلب `favicon.ico` المفقود.
4. تحسين حقل البريد في شاشة الدخول باستخدام `autocomplete="username"`.
5. تحديث روابط الكاش إلى `v=163`.

ملاحظة:
بعد تسجيل الدخول بحساب نشط تعود مزامنة Supabase للعمل كالمعتاد.
