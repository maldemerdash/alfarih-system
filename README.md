# نسخة v163 - إصلاح أخطاء التشغيل

تم البناء على نسخة v162 مع إبقاء بنية التشغيل نفسها:

- `index.html`
- `style.css`
- `app.js`
- ملفات Supabase وملفات الشرح المرافقة

أهم ما تم تعديله:

- تنظيف `app.js` بتنسيقه آليًا وفك ضغط الكتل المضغوطة ليصبح قابلًا للقراءة والمراجعة.
- دمج إصلاحات v161 داخل `app.js` بدل تحميل ملف إصلاح جانبي.
- منع محاولات الحفظ إلى Supabase قبل تسجيل الدخول حتى لا تظهر أخطاء 401/RLS في الكونسول.
- إضافة favicon من ملف `sar-symbol.png` لمنع خطأ `favicon.ico` 404.
- إضافة دعم أيقونتي `refresh` و`plane` اللتين كانتا مستخدمتين في الواجهة وغير معرفتين في قاموس الأيقونات.
- تحويل تاريخ لوحة التحكم وعنوان مسير الرواتب من نصوص ثابتة إلى قيم ديناميكية حسب تاريخ الجهاز/توقيت الرياض.
- تحديث روابط الكاش إلى `v=163`.
- إصلاح Edge Function حتى تحفظ الأدوار `admin`, `hr`, `accountant`, `manager`, `employee` بدل تحويل كل دور غير `admin` إلى `employee`.
- تشديد SQL الخاص بجدول `app_user_profiles` بإضافة أعمدة الربط والصلاحيات والفهارس وسياسة إدارة مبنية على مدير نشط.
- تحديث ملفات التشغيل والتوثيق القديمة التي لم تعد تصف النسخة الحالية بدقة.

ملاحظات مهمة:

- تم تجنب تفكيك `app.js` إلى ملفات متعددة لأن الملف يحتوي سلسلة تصحيحات تعتمد على ترتيب التنفيذ. بدل ذلك تم تنظيفه وتنسيقه ودمج آخر إصلاحات v161 في نهايته.
- لا يتم حذف أي بيانات من المتصفح أو Supabase بمجرد استبدال الملفات.
- ملفات SQL الاختيارية التي تحذف أو تصفر بيانات مالية ما زالت اختيارية ولا تشغل إلا عند الحاجة.
