# نسخة v161 - مراجعة وتنظيف عامة

تم البناء على نسخة v160 مع إبقاء بنية التشغيل نفسها:

- `index.html`
- `style.css`
- `app.js`
- `app-v161-fixes.js`
- ملفات Supabase وملفات الشرح المرافقة

أهم ما تم تعديله:

- إضافة ملف `app-v161-fixes.js` لعزل إصلاحات v161 بدل زيادة تكدس `app.js`.
- إضافة دعم أيقونتي `refresh` و`plane` اللتين كانتا مستخدمتين في الواجهة وغير معرفتين في قاموس الأيقونات.
- تحويل تاريخ لوحة التحكم وعنوان مسير الرواتب من نصوص ثابتة إلى قيم ديناميكية حسب تاريخ الجهاز/توقيت الرياض.
- تحديث روابط الكاش إلى `v=161`.
- إصلاح Edge Function حتى تحفظ الأدوار `admin`, `hr`, `accountant`, `manager`, `employee` بدل تحويل كل دور غير `admin` إلى `employee`.
- تشديد SQL الخاص بجدول `app_user_profiles` بإضافة أعمدة الربط والصلاحيات والفهارس وسياسة إدارة مبنية على مدير نشط.
- تحديث ملفات التشغيل والتوثيق القديمة التي لم تعد تصف النسخة الحالية بدقة.

ملاحظات مهمة:

- تم تجنب تفكيك `app.js` بالكامل لأن الملف يحتوي سلسلة تصحيحات تعتمد على ترتيب التنفيذ. تم فصل إصلاحات v161 في ملف مستقل آمن التحميل بعده.

- ملفات SQL الاختيارية التي تحذف أو تصفر بيانات مالية ما زالت اختيارية ولا تشغل إلا عند الحاجة.
