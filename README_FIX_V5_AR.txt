نسخة v5 - إصلاح آمن بعد مشكلة v4

المبدأ:
- تم الرجوع إلى قاعدة v3 المستقرة، وعدم لمس منطق الصلاحيات أو القوائم.
- التعديلات محصورة في: تحميل المرفقات، شكل زر التحميل، وصورة الموظف أعلى الموقع.

ما تم تعديله:
1) إصلاح مشكلة عدم تحميل الموقع/اختفاء القوائم التي ظهرت بعد v4 بالرجوع إلى منطق v3 المستقر.
2) زر وثائق المنشأة أصبح أيقونة تحميل فقط، والتحميل مباشر عبر Blob بدون فتح نافذة منبثقة.
3) مرفقات الموظف تظهر أيقونة التحميل بجوار زر الإرفاق في نفس السطر.
4) صورة الموظف المرتبط تظهر أعلى الموقع بدون تغيير منطق الصلاحيات أو إخفاء القوائم.
5) لا يوجد SQL جديد مطلوب.
