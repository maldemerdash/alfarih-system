نسخة ربط الصلاحيات التفصيلية بكل أجزاء النظام

التحديثات:
- نقل الصلاحيات التفصيلية إلى الإعدادات > الصلاحيات والأمان.
- إضافة صلاحيات على مستوى الصفحة، البطاقة، الزر، الجدول، ونوع البيانات.
- إبقاء الأدوار فقط: مدير النظام / موظف.
- ربط زر تسجيل الغياب، مراجعة الطلبات، المسافرون، الرواتب، العمولات، الوثائق، الأقسام، وإدارة المستخدمين بصلاحيات واضحة.
- الموظف لا يرى بيانات الآخرين إلا إذا تم تفعيل صلاحيات عرض الجميع.

بعد رفع الملفات إلى GitHub:
1) شغّل SQL الموجود في SUPABASE_PERMISSIONS_MATRIX_SQL.sql داخل Supabase SQL Editor.
2) إذا كانت دالة admin-create-user محدثة مسبقًا ولا توجد مشكلة في إضافة المستخدمين، لا تحتاج تحديثها.
3) افتح الموقع بحساب المدير ثم الإعدادات > الصلاحيات والأمان واضبط صلاحيات الموظفين.
