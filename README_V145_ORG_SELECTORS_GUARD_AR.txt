نسخة v145 - تحصين الهيكل الإداري في نموذج الموظف

تم تطبيق تصحيح وقائي محدود على منطق اختيار الإدارة / القسم / المهنة في نموذج الموظف.

ما تم:
- عدم اعتماد أول إدارة تلقائيًا إذا كانت خانة الإدارة فارغة.
- عدم عرض أقسام أو مهن قبل اختيار الإدارة/القسم فعليًا.
- تفريغ القسم إذا لم يعد تابعًا للإدارة المختارة.
- تفريغ المهنة إذا لم تعد تابعة للقسم المختار.
- عدم تعيين المدير المباشر تلقائيًا إلا بعد اختيار إدارة فعلية.
- الحفاظ على الشكل والتصميم والحفظ كما هو.

لا يوجد SQL مطلوب.
