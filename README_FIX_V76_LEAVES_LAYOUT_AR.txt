نسخة v76 - إعادة هيكلة صفحة الإجازات والسفر

الملفات المطلوب رفعها على GitHub Pages:
1) app.js
2) index.html
3) style.css

التعديلات:
- حذف بطاقة رصيد الإجازات من صفحة الإجازات والسفر.
- حذف جدول طلبات السفر المستقل، ودمج كل حالات السفر داخل جدول واحد باسم السفر.
- إضافة تبويبين داخليين فقط: السفر والإجازات.
- كل تبويب يحتوي جدولًا واحدًا من أول الصفحة إلى نهاية البطاقة.
- منع السكرول الأفقي داخل الجداول نهائيًا قدر الإمكان.
- السماح بسكرول عمودي داخلي داخل بطاقة الجدول، مع ضبطه ليظهر في جهة اليسار في المتصفحات الداعمة.
- إضافة فلاتر أعلى كل جدول: الكل، بانتظار الاعتماد، معتمد/مسافر، تمت المباشرة، مرفوض.
- تحسين شاشة اعتماد السفر المنبثقة بهوية الموقع، مع خانات مرفقات مرتبة للتذكرة والتأشيرة.
- الحفاظ على منطق v75: منع التعارض، حفظ السحابة، رصيد الإجازات العام، المرفقات، العمولة، ومباشرة السفر دون تحويله لإجازة.

لا يوجد SQL جديد.
