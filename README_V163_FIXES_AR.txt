نسخة v163

تم تنفيذ الإصلاحات فوق نسخة v160 المعتمدة، مع الاستفادة من نسخة v162 السابقة كآخر عمل، ثم إزالة طبقة JavaScript الخاصة بإصلاح المرفقات في v162 لأنها كانت تتداخل مع معالجات المرفقات القديمة.

الإصلاحات:
1. تفعيل مرفقات ملف الموظف من معالج واحد أعلى من المعالجات القديمة، بحيث لا تتدخل الأكواد السابقة في حدث رفع الملف.
2. حفظ المرفق فعليًا عبر دالة saveAttachment الحالية، ثم تثبيت رقم المرفق داخل employeeFormState، ثم حفظ بيانات الموظف محليًا وسحابيًا عبر dbSaveEmployee و saveCloudStateNow عند توفرهما.
3. شمول خانات المرفقات التالية داخل ملف الموظف:
   - صورة الموظف.
   - الهوية / الإقامة.
   - الجوازات.
   - شهادة الآيبان.
   - موافقة الموظف.
   - الوثائق.
   - التوقيع.
   - البصمة.
   - مرفق العقد.
   - مرفق الإقرار / التوثيق.
4. ظهور زر تحميل بجوار زر الإرفاق عند وجود المرفق، بدون تمدد أو تشوه بصري.
5. تعديل نافذة تأكيد تمديد العقد لتكون مرتبة وبدون سكرول داخلي، مع الحفاظ على هوية الموقع.
6. الإبقاء على نافذة إضافة الإدارة / القسم / المهنة كنافذة dialog حقيقية فوق شاشة الموظف.
