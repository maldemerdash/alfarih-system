# v162 - إصلاح طبقات النوافذ ومرفقات ملف الموظف

تم تنفيذ هذه النسخة فوق v160/v161 بدون الرجوع لأي نسخة أقدم.

## المنفذ
- تحويل نافذة إضافة الإدارة/القسم/المهنة داخل ملف الموظف إلى dialog حقيقي في طبقة المتصفح العليا حتى تظهر أمام شاشة إضافة/تحرير الموظف.
- تعطيل مسار النافذة القديمة التي كانت div عاديًا وتتأثر بترتيب الطبقات خلف نافذة الموظف.
- توحيد شكل مرفقات ملف الموظف بحيث يظهر زر تحميل بجانب زر الإرفاق داخل نفس السطر بدون تمدد أو تشوه بصري.
- دعم الحفظ الفوري لمرفقات الجواز، الحسابات البنكية، الوثائق، الهوية، التوقيع، البصمة، العقد والصورة عند تغيير المرفق.
- تثبيت حفظ بيانات المرفقات في employees ثم IndexedDB ثم الحفظ السحابي المجدول في Supabase app_settings.
- جعل زر حذف سجل تمديد العقد أحمر فقط بدون التظليل الأخضر.
