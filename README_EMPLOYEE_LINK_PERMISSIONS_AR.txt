نسخة ربط الصلاحيات بالموظف

الخطوات بعد رفع الملفات إلى GitHub:
1. شغل ملف SUPABASE_EMPLOYEE_LINK_PERMISSIONS_SQL.sql في Supabase SQL Editor.
2. حدّث دالة Edge Function باسم admin-create-user بالكود الموجود في supabase/functions/admin-create-user/index.ts ثم اضغط Deploy.
3. افتح الموقع بحساب مدير النظام.
4. من إدارة المستخدمين اربط كل مستخدم بملف موظف.
5. من الإعدادات > الصلاحيات والأمان حدد ما يظهر للموظف.

مهم:
- الموظف غير المرتبط بملف موظف لا تظهر له بيانات عشوائية.
- الموظف يرى بياناته فقط إلا إذا منحه المدير صلاحية عرض الجميع.
- تمت إضافة خانة تأكيد كلمة المرور عند إضافة مستخدم.
