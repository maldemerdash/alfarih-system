تعديل v31 - منع تكرار وتداخل طلبات السفر والإجازات

الملف المعدل:
- app.js فقط

ما تم تطبيقه:
- منع إنشاء طلب سفر جديد لنفس الموظف إذا كان لديه طلب سفر قائم أو متداخل زمنياً ولم يتم إغلاقه بالمباشرة أو الرفض.
- منع اعتماد طلب سفر إذا أصبح متعارضاً مع طلب سفر قائم لنفس الموظف.
- منع إنشاء طلب إجازة جديد لنفس الموظف إذا كان لديه طلب إجازة قائم أو متداخل زمنياً ولم يتم إغلاقه بالمباشرة أو الرفض.
- منع اعتماد طلب إجازة إذا أصبح متعارضاً مع طلب إجازة قائم لنفس الموظف.

لم يتم تعديل:
- style.css
- index.html
- قاعدة البيانات
- التصميم أو الهوية البصرية

لا يوجد SQL مطلوب.
