إصلاح v20 للمرفقات:
- معالجة اختفاء زر تحميل المرفق بعد تسجيل الخروج والدخول مرة أخرى.
- إذا كان المرفق موجودًا في جدول attachments لكن معرفه غير محمل داخل سجل الموظف، يتم استرجاعه حسب related_table أو storage_path وربطه مرة أخرى بالموظف.
- عند رفع مرفق لموظف موجود يتم حفظ معرف المرفق مباشرة داخل سجل الموظف.
- عند وجود attachmentId يتم إجبار زر التحميل على الظهور بجانب كلمة الإرفاق.
- عند فتح موظف جديد يبقى زر التحميل مخفيًا إذا لم يوجد مرفق.
لا توجد أوامر SQL مطلوبة.
