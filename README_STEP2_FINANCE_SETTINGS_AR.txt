تعديل الخطوة الثانية: إعدادات المالية

ما تم تنفيذه:
- إضافة قسم جديد داخل الإعدادات باسم: المبالغ المالية.
- إضافة حقلي: المبلغ الافتتاحي ومبلغ العهدة.
- ربط بطاقة المبلغ المرحل بقيمة المبلغ الافتتاحي.
- ربط بطاقة مبلغ العهدة بقيمة مبلغ العهدة.
- حفظ القيم في التخزين المحلي وضمن حالة التطبيق السحابية الحالية app_settings إن كانت مفعلة.

ملاحظة مهمة:
هذا التعديل لا ينفذ منطق إغلاق اليوم ولا يجمد الأيام السابقة بعد، بل يجهز إعدادات المالية وربطها بالبطاقات فقط حسب خطوة واحدة.

Supabase:
لا يوجد كود SQL جديد مطلوب لهذه الخطوة.
