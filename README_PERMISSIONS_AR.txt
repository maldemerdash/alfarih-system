خطوات تفعيل الصلاحيات التفصيلية:

1) ارفع ملفات هذه النسخة إلى GitHub ثم انتظر تحديث Vercel.
2) افتح Supabase ثم SQL Editor وشغل ملف:
   SUPABASE_GRANULAR_PERMISSIONS_SQL.sql
3) افتح Edge Functions ثم دالة admin-create-user ثم Code.
4) انسخ محتوى:
   supabase/functions/admin-create-user/index.ts
   والصقه في ملف index.ts للدالة ثم اضغط Deploy.
5) ادخل بحساب مدير النظام وافتح إدارة المستخدمين.
6) عند إضافة أو تعديل موظف ستجد جداول صلاحيات مفصلة لكل قسم.

ملاحظات:
- الأدوار أصبحت: مدير النظام / موظف فقط.
- مدير النظام له كل الصلاحيات تلقائياً.
- الموظف يرى فقط ما تسمح به في جدول الصلاحيات.
- شاشة تسجيل الدخول أصبحت بخلفية مستقلة ولا تظهر النظام خلفها.
