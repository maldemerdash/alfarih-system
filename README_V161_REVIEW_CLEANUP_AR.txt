نسخة v161 - مراجعة وتنظيف عامة
=============================

تمت مراجعة الحزمة كاملة، مع التركيز على الملفات التنفيذية والملفات التي تؤثر على التشغيل:
- index.html
- style.css
- app.js
- app-v161-fixes.js
- supabase/functions/admin-create-user/index.ts
- ملفات SQL الخاصة بالمستخدمين والصلاحيات
- ملفات README التشغيلية

الإصلاحات المنفذة:
1. فصل إصلاحات v161 في ملف مستقل `app-v161-fixes.js` حتى لا يزيد تكدس ملف `app.js`.
2. إصلاح الأيقونات الناقصة `refresh` و`plane`.
3. تحديث تاريخ لوحة التحكم ومسير الرواتب ليعتمدا على التاريخ الحالي بدل نصوص يونيو 2026 الثابتة.
4. تحديث روابط الملفات في `index.html` إلى `v=161`.
5. إصلاح دالة `admin-create-user` حتى تحفظ كل الأدوار المدعومة ولا تحولها إلى `employee`.
6. تشديد سياسات `app_user_profiles` في ملفات SQL الأساسية، مع إضافة أعمدة `employee_id` و`permissions` والفهارس اللازمة.
7. تحديث README القديم وملف التشغيل العربي ليتطابقا مع النسخة الحالية.

لماذا لم يتم تفكيك `app.js` بالكامل؟
لأن الملف الحالي يحتوي سلسلة طويلة من تصحيحات الإصدارات السابقة، وكثير منها يلف دوال سابقة حسب ترتيب التحميل. فصل هذا الملف جذريا بدون اختبارات وظيفية كاملة على قاعدة بيانات حقيقية قد يكسر ترتيب التنفيذ. لذلك تم اختيار فصل إصلاحات v161 الجديدة في ملف مستقل يحمل بعد `app.js`.

طريقة الاستبدال:
استبدل ملفات النسخة الحالية بمحتويات هذه الحزمة كاملة، وليس الملفات المعدلة فقط.
