# تعديل v142 - فتح تحرير الموظف من إشعارات الجرس

تم تعديل سلوك الضغط على إشعارات الموظف داخل قائمة جرس الإشعارات.

## التعديل
- إشعارات الإقامة/الهوية، الجواز، العقد، رخصة العمل، ووثائق الموظف تفتح شاشة تحرير بيانات الموظف مباشرة.
- لم تعد تفتح معاينة الموظف كما لو تم الضغط على علامة العين.
- إشعارات الإجازات والسفر ما زالت تفتح شاشة الإجازات والسفر.
- إشعارات الغياب بدون موظف محدد تفتح الحضور والانصراف.
