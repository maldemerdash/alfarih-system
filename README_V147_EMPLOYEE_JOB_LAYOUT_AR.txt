نسخة v147 - ترتيب البيانات الوظيفية وتصحيح المدير المباشر

تم البناء على نسخة v146، وتم تعديل نطاق محدد فقط:

1. ترتيب حقول البيانات الوظيفية داخل نموذج الموظف:
   - الفرع أولاً.
   - الإدارة بعد الفرع.
   - القسم بعد الإدارة.
   - المهنة بعد القسم.
   - المدير المباشر في السطر التالي.

2. إضافة أزرار/روابط سريعة تحت الحقول:
   - إضافة إدارة.
   - إضافة قسم.
   - إضافة مهنة.
   وتفتح قسم الأقسام/الهيكل الإداري من الإعدادات.

3. تصحيح تعبئة المدير المباشر:
   - لا يتم تعيين مدير قبل اختيار إدارة فعلية.
   - إذا اختير قسم وله مدير، يعتمد مدير القسم.
   - إذا لم يكن للقسم مدير، يعتمد مدير الإدارة.
   - يدعم بنية الهيكل القديمة وبنية v136 الحالية.

4. صورة الموظف في عرض جميع الموظفين:
   - الضغط على اسم الموظف يفتح التحرير كما هو.
   - الضغط على صورة الموظف يفتح شاشة العرض مثل أيقونة العين.

لم يتم تعديل الصلاحيات أو الإشعارات أو المالية أو التقارير أو المرفقات.
لا يوجد SQL جديد مطلوب.
