نسخة v148 - تنظيف منطق البيانات الوظيفية والحضور والرواتب

تم تنفيذ التعديلات التالية فقط:
1. تنظيف ظهور شاشة الأقسام / الهيكل الإداري واعتماد الواجهة الحديثة مباشرة.
2. إعادة حقول الفرع / الإدارة / القسم / المهنة إلى أول تبويب البيانات الوظيفية.
3. إضافة منبثقة داخل ملف الموظف لإضافة إدارة / قسم / مهنة دون الانتقال إلى الإعدادات.
4. ربط إضافة المهنة بالقسم المختار في ملف الموظف، وربط إضافة القسم بالإدارة المختارة.
5. جعل صورة الموظف في جدول الموظفين تفتح شاشة العرض مثل أيقونة العين، مع بقاء الاسم والتحرير كما هما.
6. عند إدخال تاريخ بداية العقد لموظف جديد بدون رقم موظف يتم تعبئة تاريخ المباشرة تلقائيا مع بقائه قابلا للتحرير.
7. منع ظهور الموظف في الحضور والانصراف قبل تاريخ المباشرة.
8. منع ظهور الموظف في مسير الرواتب قبل تاريخ المباشرة.

لا يوجد SQL جديد مطلوب.
