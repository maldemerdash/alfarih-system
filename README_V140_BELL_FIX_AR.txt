# v140 - تصحيح نهائي لإشعارات جرس الهيدر

تمت مراجعة منطق الجرس كأنه لا يعمل، وتمت إضافة طبقة نهائية دقيقة تخص جرس الإشعارات فقط.

ما تم:
- منع أحداث الجرس القديمة التي كانت تعرض Toast أسفل الشاشة بدل فتح القائمة.
- اعتماد مصدر الوثائق المنتهية وقريبة الانتهاء بنفس منطق بطاقات الصفحة الرئيسية للموظفين: الإقامة/الهوية، الجوازات، ووثائق الموظف.
- الإشعارات تظهر داخل قائمة الجرس فقط.
- تحسين شكل زر الجرس والعداد فوقه.
- ضبط القائمة المنسدلة بحيث تظهر في الهيدر ولا تتعارض مع الصفحة.
- لا يوجد SQL جديد.

الملفات المعدلة:
- app.js
- index.html
