# v139 - مراجعة وتصحيح إشعارات جرس الهيدر

تمت مراجعة منطق قائمة الجرس كأنها لا تعمل، وتمت إضافة طبقة تصحيح خفيفة تخص الجرس فقط.

التصحيح يشمل:
- إعادة بناء القائمة عند فتح الجرس وعند التحديث.
- قراءة إعدادات الإشعارات نفسها المستخدمة في شاشة الإعدادات.
- دعم حقول التواريخ الفعلية مثل identityExpiryGregorian إضافة إلى الحقول العامة.
- تحسين فتح التنبيه المرتبط بموظف أو قسم الإجازات أو الحضور.
- عدم إضافة أي إشعارات إلى الصفحة الرئيسية.
- عدم تغيير منطق الصلاحيات أو إعداد العمل.
- عدم إضافة SQL جديد.

الملفات المعدلة:
- app.js
- index.html
