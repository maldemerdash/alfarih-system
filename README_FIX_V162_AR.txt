إصلاح v162 - زر حذف الموظف

النسخة الأساسية:
نسخة_تنظيف_احترافية_وازالة_الطبقات_القديمة_v161.zip

المشكلة:
زر حذف الموظف لا ينفذ الحذف بعد تنظيف الطبقات القديمة.

الإصلاح:
- إضافة طبقة حماية صغيرة ومحددة للحذف فقط.
- جعل زر حذف الموظف type="button" لمنع أي سلوك submit غير مقصود.
- التقاط زر الحذف وفتح نافذة التأكيد بشكل ثابت.
- عند تأكيد الحذف يتم حذف الموظف من:
  1) الذاكرة الحالية employees
  2) IndexedDB
  3) localStorage / nawah-employees
  4) الإجازات المرتبطة
  5) سجلات الحضور الاستثنائية المرتبطة
  6) السلفيات/الرواتب المرتبطة عند توفر دالة التنظيف
- تنفيذ حفظ سحابي فوري saveCloudStateNow({force:true}) بعد الحذف حتى لا يرجع الموظف بعد التحديث أو تسجيل الدخول.
- لم يتم تغيير التصميم أو الواجهة.
