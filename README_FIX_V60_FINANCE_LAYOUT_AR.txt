نسخة v60 - ترتيب بطاقات المالية

تم تنفيذ التعديل المطلوب في شاشة المالية فقط:

1. حذف بطاقة "إجمالي المبالغ المعلقة" من صف بطاقات الملخص العلوي لأنها موجودة في رأس جدول المبالغ المعلقة.
2. ترحيل بطاقات الصف الثاني بدون ترك فراغ.
3. نقل بطاقة "مبلغ الصندوق" إلى أقصى اليسار تحت بطاقة "المبلغ المرحل الجديد".
4. نقل زر "إغلاق اليوم" إلى شريط التحكم العلوي بجوار زر "إدخال سلفة".
5. الحفاظ على نفس معادلات المالية ومنطق الحفظ والسرعة، بدون أي تعديل على قاعدة البيانات.

الملفات التي يجب رفعها على GitHub Pages:
- index.html
- app.js
- style.css

لا يوجد SQL جديد.
