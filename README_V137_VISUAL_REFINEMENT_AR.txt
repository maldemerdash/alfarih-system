# نسخة v137 - تحسين الواجهات وتنظيف عناصر الإعدادات

تم تنفيذ تعديلات واجهة فقط على نسخة v136:

1. تحسين تصميم الحساب الشخصي بتوزيع بطاقات احترافية متوافقة مع هوية الموقع.
2. تحسين تصميم شاشة الإشعارات دون تنفيذ منطق إشعارات فعلي.
3. إضافة جرس إشعارات في الهيدر بين صورة الموظف وزر الخروج مع قائمة منسدلة جاهزة للبرمجة لاحقًا.
4. تعديل زر الخروج ليكون أيقونة حمراء فقط بدون دائرة.
5. إزالة البطاقة الخضراء التوضيحية من الصلاحيات والأمان.
6. إزالة بطاقة مراجعة إعداد العمل الخضراء.
7. ترتيب بطاقات قاعدة بيانات الغياب الأربع في سطر واحد مع تصغير الخط والمسافات.

لا يوجد SQL جديد مطلوب لهذه النسخة.
