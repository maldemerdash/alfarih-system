إصلاح v170 - نافذة تأكيد حذف التنبيه الخاص
==========================================

المشكلة:
- عند حذف تنبيه خاص كان النظام يستخدم رسالة المتصفح الافتراضية `confirm`.
- هذه الرسالة تظهر خارج هوية الموقع وبشكل مختلف حسب المتصفح والنطاق.

ما تم إصلاحه:
- تم إنشاء نافذة تأكيد داخلية باسم `privateAlertDeleteConfirmModal`.
- النافذة تستخدم نفس نمط نوافذ النظام وأزراره.
- زر الإلغاء يغلق النافذة فقط.
- زر الحذف ينفذ نفس منطق الحذف السابق، وينقل التنبيه إلى جدول التنبيهات المحذوفة مع حفظ بيانات الحذف.

الملفات التي تم تعديلها:
- `app.js`
- `style.css`
- `index.html`

- `README_RUN_AR.txt`
