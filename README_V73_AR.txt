نسخة v73 - منع تكرار وتعاض طلبات السفر والإجازات

الهدف:
- منع إنشاء طلب سفر مكرر لنفس الموظف إذا كان لديه طلب سفر/إجازة قائم أو بانتظار الموافقة في نفس الفترة.
- منع إنشاء طلب إجازة مكرر لنفس الموظف إذا كان لديه طلب إجازة/سفر قائم أو بانتظار الموافقة في نفس الفترة.
- منع اعتماد طلب سفر أو إجازة إذا كان هناك طلب آخر لنفس الموظف في نفس الفترة ولم يتم رفضه أو إنهاؤه/المباشرة منه.

ما لم يتغير:
- لا يوجد SQL جديد.
- لم يتم تغيير قاعدة البيانات.
- لم يتم تغيير دورة اعتماد السفر أو شاشة التذكرة والتأشيرة.
- لم يتم تغيير رصيد الإجازات أو الخصم عند الاعتماد.

الرفع على GitHub Pages:
- ارفع app.js و index.html فقط.
