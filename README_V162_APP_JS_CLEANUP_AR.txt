نسخة v162 - مراجعة وتنظيف عامة
=============================

تمت مراجعة الحزمة كاملة، مع التركيز على الملفات التنفيذية والملفات التي تؤثر على التشغيل:
- index.html
- style.css
- app.js
- supabase/functions/admin-create-user/index.ts
- ملفات SQL الخاصة بالمستخدمين والصلاحيات
- ملفات README التشغيلية

الإصلاحات المنفذة:
1. تنظيف `app.js` بتنسيقه آليًا وفك ضغط الكتل المضغوطة ليصبح قابلًا للقراءة والمراجعة.
2. دمج إصلاحات v161 داخل `app.js` بدل تحميل ملف جانبي.
3. إصلاح الأيقونات الناقصة `refresh` و`plane`.
4. تحديث تاريخ لوحة التحكم ومسير الرواتب ليعتمدا على التاريخ الحالي بدل نصوص يونيو 2026 الثابتة.
5. تحديث روابط الملفات في `index.html` إلى `v=162`.
6. إصلاح دالة `admin-create-user` حتى تحفظ كل الأدوار المدعومة ولا تحولها إلى `employee`.
7. تشديد سياسات `app_user_profiles` في ملفات SQL الأساسية، مع إضافة أعمدة `employee_id` و`permissions` والفهارس اللازمة.
8. تحديث README القديم وملف التشغيل العربي ليتطابقا مع النسخة الحالية.

لماذا لم يتم تفكيك `app.js` بالكامل؟
لأن الملف الحالي يحتوي سلسلة طويلة من تصحيحات الإصدارات السابقة، وكثير منها يلف دوال سابقة حسب ترتيب التحميل. فصل هذا الملف جذريا بدون اختبارات وظيفية كاملة على قاعدة بيانات حقيقية قد يكسر ترتيب التنفيذ. لذلك تم اختيار تنظيف الملف الحالي وتنسيقه ودمج إصلاحات v161 في نهايته كآخر طبقة تشغيل.

طريقة الاستبدال:
استبدل ملفات النسخة الحالية بمحتويات هذه الحزمة كاملة، وليس الملفات المعدلة فقط.
